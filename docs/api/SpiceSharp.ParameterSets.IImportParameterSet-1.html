Interface IImportParameterSet<P>

An IParameterSet that supports importing or setting parameters by specifying the name of the parameter. All parameters are of the same type.

Type Parameters
Name Description
P

The type of the parameter values.
Remarks

Parameters can be named using the ParameterNameAttribute. This interface can be used to avoid reflection and speed up setting parameters, but the code base increases.
