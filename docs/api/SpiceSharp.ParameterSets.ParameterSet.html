Class ParameterSet

The default implementation for a IParameterSet. It uses reflection as a last line of defense.

public abstract class ParameterSet : IParameterSet
Remarks

This class will use the IExportPropertySet<P> or IImportParameterSet<P> if they are defined on the class to avoid reflection. If it isn't defined, it will fall back to reflection.
